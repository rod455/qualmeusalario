Qual Meu Salário?

Política de Privacidade

Última atualização: 15 de março de 2025

Resumo: não coletamos dados salariais identificáveis, não vendemos dados para terceiros e você pode deletar sua conta a qualquer momento.

1. Quem somos

O aplicativo Qual Meu Salário? é desenvolvido e operado de forma independente, com sede no Brasil. Para dúvidas, entre em contato: contato@qualmeusalario.com.br

2. Dados que coletamos

Coletamos apenas o mínimo necessário para o funcionamento do app:

  • Dados de análise salarial — cargo, cidade, modelo de trabalho, faixa salarial e anos de experiência. Esses dados são armazenados de forma anônima quando você não tem conta e vinculados à sua conta quando logado.
  • Dados de conta (opcional) — nome, e-mail e foto de perfil, fornecidos via Google Sign-In ou cadastro manual.
  • Dados de uso — telas visitadas, tempo de sessão e erros técnicos, coletados de forma agregada e anônima para melhorar o app.

3. O que NÃO coletamos

  • CPF, RG, dados bancários ou informações financeiras além do salário declarado voluntariamente
  • Contatos, câmera, microfone ou localização precisa
  • Dados de outros apps instalados no dispositivo

4. Como usamos os dados

  • Calcular o benchmark salarial para o seu cargo e cidade
  • Salvar seu histórico de análises (somente se você tiver conta)
  • Enviar notificações push sobre variações do mercado (somente com sua autorização)
  • Melhorar a precisão dos dados de mercado de forma agregada

5. Compartilhamento de dados

Não vendemos, alugamos nem compartilhamos seus dados pessoais com terceiros para fins comerciais. Compartilhamos apenas com:

  • Supabase (banco de dados) — armazenamento seguro dos dados de conta e análises
  • Google AdMob — exibição de anúncios. O AdMob pode coletar identificadores de publicidade do dispositivo conforme sua política de privacidade
  • Autoridades legais — somente se exigido por lei

6. Anúncios (AdMob)

O app exibe anúncios fornecidos pelo Google AdMob. O Google pode usar o identificador de publicidade do seu dispositivo (GAID no Android) para personalizar anúncios. Você pode optar por não receber anúncios personalizados nas configurações do seu dispositivo em:

Configurações → Google → Anúncios → Cancelar personalização de anúncios

7. Retenção de dados

  • Análises sem conta: armazenadas localmente no dispositivo, deletadas ao desinstalar o app
  • Análises com conta: mantidas enquanto a conta existir
  • Dados de conta: deletados em até 30 dias após a exclusão da conta

8. Seus direitos (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (Lei 13.709/2018), você tem direito a:

  • Acessar os dados que temos sobre você
  • Corrigir dados incorretos
  • Solicitar a exclusão dos seus dados
  • Revogar o consentimento para coleta de dados
  • Portabilidade dos dados

Para exercer esses direitos, entre em contato: contato@qualmeusalario.com.br

9. Segurança

Utilizamos criptografia em trânsito (HTTPS/TLS) e em repouso para todos os dados armazenados. O acesso ao banco de dados é protegido por Row Level Security (RLS), garantindo que cada usuário acesse apenas seus próprios dados.

10. Crianças

O app é destinado a pessoas com 18 anos ou mais. Não coletamos intencionalmente dados de menores de idade. Se você identificar que um menor forneceu dados, entre em contato para que possamos removê-los.

11. Alterações nesta política

Podemos atualizar esta política periodicamente. Notificaremos sobre mudanças significativas via notificação push ou e-mail. O uso continuado do app após as alterações indica aceitação da nova política.

12. Contato

Dúvidas, solicitações ou reclamações sobre privacidade:
